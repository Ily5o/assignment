LISTS EXPANDED
  • orderd lists
  • unorderd lists
  • description lists

orderd lists

There used for items that occur in a particular order such as step-by-step instructins. They use the ol element (ordered lists), and while using the ordered lists the browser automatically insersts numbers before the items hence you dont need to add them

UNORDERD LISTS

They are used to make a list of different items such as names,places,roads and options.Its identified using the ul element, unorderd lists use bullets before its items

DESCRIPTION LISTS

Are used for any type of terms and their descriptions,the description lists are more different than orderd listsand unordered lists. In description lists the dl element describes the entire description, the dt element is used to show the name and the dd element gives the description
